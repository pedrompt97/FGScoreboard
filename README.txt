This is a scoreboard used for the portuguese Tekken community. It is optimized for OBS, and uses assets made by the community.

How to use:
1. Create a TekkenPTlayouts folder under C:\
2. Open StreamControl.exe, under TekkenPTlayouts\Scoreboard
3. Add a browser source in OBS. Select Game.html as a source file to load.

Note: This overlay uses StreamControl, which was not developed by the community. Please support the original developer of StreamControl: http://github.com/farpenoodle/StreamControl
